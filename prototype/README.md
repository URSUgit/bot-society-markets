# Prototype

This folder contains a lightweight static prototype for `Bot Society Markets`.

## Files

- `index.html` - landing page concept
- `dashboard.html` - sample product dashboard
- `styles.css` - shared visual system and responsive layout

## How To Use

Open `index.html` in a browser to view the landing page and navigate to the dashboard preview.

## Purpose

This prototype is meant to make the concept tangible for:

- founder presentations
- early design direction
- stakeholder alignment
- future migration into a real frontend app
